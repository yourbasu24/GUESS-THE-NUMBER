Number Guessing Game
This is a simple number guessing game implemented in Python using the tkinter library for the graphical user interface (GUI). The game allows the player to guess a randomly generated number within a specified range. It also provides features such as choosing difficulty levels, receiving hints, and changing the background color of the GUI.
Features
Choose difficulty levels: Easy, Medium, or Hard.
Guess the number within a specified range.
Receive hints to help narrow down the correct answer.
Change the background color of the GUI.
Requirement
* Python 3.12
* tkinter library
* PIL (Python Imaging Library) for handling images
* pyttsx3 library for text-to-speech functionality
Gameplay
Enter your name in the provided input box.
Click on the "Submit" button to proceed.
Choose the difficulty level: Easy, Medium, or Hard.
Guess the number within the specified range.
Use the "Hint" button to receive hints if needed.
Try to guess the number within the allotted number of attempts.
Receive feedback on your guesses and the final outcome of the game.
